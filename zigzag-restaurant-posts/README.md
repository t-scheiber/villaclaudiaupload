# ZigZag Restaurant Posts

A WordPress plugin that allows you to display your restaurant posts in an attractive zig-zag layout that alternates between left and right image placement.

## Features

- Beautiful responsive zig-zag layout for restaurant posts
- Works with the WordPress Block Editor (Gutenberg)
- Works with Divi Builder
- Works as a shortcode for maximum flexibility
- Displays featured image, title, excerpt, and "Learn More" button
- Customizable number of posts and category filtering
- Fully responsive design (looks great on all devices)

## Installation

1. Download the plugin zip file
2. Go to your WordPress admin dashboard → Plugins → Add New
3. Click "Upload Plugin" and select the zip file
4. Activate the plugin

## Usage

### In Block Editor (Gutenberg)

1. Add a new block and search for "ZigZag Restaurants"
2. Customize the settings in the block sidebar
3. Save your page or post

### In Divi Builder

1. Add a new module and search for "ZigZag Restaurants"
2. Customize the settings in the module settings panel
3. Save your page or post

### Using Shortcode

You can also use the shortcode in any text widget or page builder:

```
[zigzag_restaurants category="italian" posts_per_page="4" class="my-custom-class"]
```

#### Shortcode Parameters

- `category`: (optional) Category slug to filter posts
- `posts_per_page`: (optional) Number of posts to display (default: 6)
- `class`: (optional) Custom CSS class for additional styling

## Styling

The plugin comes with a clean, responsive design out of the box. You can add your own custom CSS to further style the layout if needed.

## Requirements

- WordPress 5.0 or higher
- Works with any modern theme
